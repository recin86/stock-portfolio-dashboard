# 🏦 포트폴리오 대시보드

React + TypeScript + Vite로 만든 증권 포트폴리오 대시보드

## 🚀 실행 방법
```bash
npm install
npm run dev
```

## ✨ 기능
- 4개 계좌 관리 (본인/배우자 일반계좌 + ISA)
- 실시간 포트폴리오 요약 및 차트 시각화
- 종목별 비중 Top5 도넛 차트
- 계좌별 자산 분포 차트
- 계좌별 상세 정보 페이지
- 한국식 상승(빨간색)/하락(파란색) 색상
- Pretendard 폰트 적용
- 토스뱅크 스타일 디자인

## 🛠 기술 스택
- React + TypeScript + Vite
- Chart.js + react-chartjs-2
- React Router
- Lucide React Icons
